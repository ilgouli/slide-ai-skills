# table — 表格

```yaml
type: table
heads: [列1, 列2, 列3]
widths: ["40%", "30%", "30%"]   # 可选，列宽
rows:
  - [单元格, 单元格, 单元格]
footer: "备注文字"               # 可选
```

## 参数说明

| 字段 | 类型 | 说明 |
|------|------|------|
| `heads` | string[] | 表头，必填 |
| `widths` | string[] | 列宽百分比，可选 |
| `rows` | string[][] | 数据行，必填 |
| `footer` | string | 底部备注，可选 |
| `left_border` | string | 左侧竖线颜色 |

## 示例

```yaml
type: table
heads: [模块, 负责人, 状态, 上线日期]
widths: ["30%", "20%", "20%", "30%"]
rows:
  - [规则引擎, 张三, ✅ 已上线, 2026-05-01]
  - [执行引擎, 李四, 🚧 开发中, 2026-07-15]
  - [监控面板, 王五, 📋 规划中, TBD]
footer: "数据截至 2026-07-01"
```
