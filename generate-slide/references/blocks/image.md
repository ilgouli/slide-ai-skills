# image — 图片

```yaml
type: image
src: "/img/screenshot.png"
caption: "图片说明"       # 可选
```

## 参数说明

| 字段 | 类型 | 说明 |
|------|------|------|
| `src` | string | 图片路径或 URL，必填 |
| `caption` | string | 图片说明文字，可选 |

## 示例

```yaml
type: image
src: "/decks/demo/arch-diagram.png"
caption: "图1：系统整体架构"
```
